# LeadBot Pro - AI-Powered Lead Qualification

## What is this?
An intelligent chatbot that talks to potential customers, understands their needs, and saves their information automatically.

## Features
- 🤖 Local AI (no API costs, runs offline)
- 💾 Automatic lead extraction and storage
- 📊 Lead scoring and qualification
- 🌐 Web interface and REST API

## Quick Start
1. Install Ollama and pull mistral model
2. Run: `pip install -r requirements.txt`
3. Run: `python -m app.main`
4. Open: http://localhost:8000

## Tech Stack
- FastAPI (Backend)
- Ollama + Mistral (Local LLM)
- SQLite (Database)
- HTML/CSS/JS (Frontend)
